Welcome to Swap Bot (0.0.2)

Made for swapping rims on locked tune files for Forza Horizon 5, but also contains a handful of game commands and more from an old Kik bot. This will be familiar to some as PiBot.

How to use

This bot was made for rim swapping locked tune files, but has a few other fun commands and more being worked on daily. The tutorial below is also available with the /swap:help command.

To start replacing those dirty rims, you need to find the directory your tunes are saved in.

Finding your tune file

If you play on the Xbox (Microsoft) version
Location: C:/Users/[USER]/AppData/Local/Packages/Microsoft.624F8B84B80_8wekyb3d8bbwe/SystemAppData/wgs/[XUID]/

(You may have 3+ folders (ignore the t folder) in your wgs directory if you have multiple accounts signed in.) If you don't know which XUID is yours, it should be the most recently modified folder (after you download a tune). Once you are in this folder, you should see a bunch of hashed folder names and a container file. Open up the most recently created/modified folder in here as well. You should see a few 1kb files and another container file. Hover over each file, and find the one that is 378 bytes. This is the file you want to attach to the !swap command.

If you play on the Steam version
Location: C:/Program Files (x86)/Steam/userdata/[STEAMID]/155136/remote/[XUID]/

Steam is a lot simpler. Download your tune and open up the most recently modified directory in the remote folder. You should see names along the lines of Tuning_####[date]####.Data as well as ones with .header and thumbnails. You want the one that ends in .Data. Make sure it's the tune you downloaded by setting it to recently created/modified.

How to find the RIM ID

To find the RIM ID of the rim you want to apply, go to RIM IDS and ctrl+f to search for your rim. The exact name won't be there so try searching parts of it, or just scroll through it as it's ordered by manufacturer.
When using the /swap command, you only need the RIM_ID portion of the text. For example, the list gives me WOR_XD9 (XD9), I would only need to type /swap WOR_XD9

How to send the tune file

To send the tune file to the bot, attach it the /swap [RIM_ID] command you built using the above steps. After sending the message, the bot will reply with a message displaying info like the car name, original rims, and new rims, and the updated tune file that you will replace the old one with.



All commands

All available commands, as well as future commands, WIP commands, and behind the scenes features. If you would like to suggest a feature or minigame, feel free to DM me and I'll be glad to add it.
  • /help ----- Displays help commands
  • /ping ----- Tests discord response time (helps calculate more accurate !type score)
  • /roll [your question] ----- Rolls a number between 11111111 and 99999999. The more duplicates the number ends with, the higher the likeliness your statement is true.
    Example: !roll Am I amazing?
    Swap Bot: [username]> Am I amazing? ...66345888 Triples!!!
    The lowest is singles, (most common) and the highest is Octuples (least common)
  • /shouldi [something you wanna/don't wanna do] ----- Randomized replies to help you decide on something if you're feeling indecisive. May or may not have programmed responses for certain keywords. :)
  • /8balls [8ball question] ----- Another randomized response command, but in the 8ball format.
  • !guess:[easy | normal | hard] ----- Guess a number between 15, 25, or 50 with !guess:easy, normal, easy, or hard mode. Example: !guess:easy 12
  • /type ----- Gives you a message to type as fast as you can. Fastest responses are stored, and you'll be notified when you set a record.
  • /type:score ----- Returns your personal record
  • /type:score:global ----- Returns fastest typer so far
  • /swap [RIM_ID] [tuneFile] ----- Swap rims for ones of your choice using the provided tune file.
  • /swap:help ----- Displays help command for swapping rims.
  • /swap:rims ----- Replies with link to RIM IDS (link here)

    • Commands coming in 0.1.0

  • /rps [rock | paper | scissors] ----- Play the bot in a game of rock, paper, scissors.
  • /fish ----- Another old and simple but fun game from the Kik messenger era. Pick your permanent perks, go fishing, and sell your earnings for more perks and upgrades. The minigames will be attached to the currency system eventually.
